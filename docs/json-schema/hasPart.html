hasPart

Type: array

If the dataset is a collection of smaller datasets, use the hasPart property to denote such relationship. Can take the form of an array of URL(s) or Dataset objects.

Each item of this array must be:

  • URI.schema.json
  • Option 2
Type: string

String representing a URI.
Type: object

Example:

[
    {
        "@type": "Dataset",
        "name": "Sub dataset 01",
        "description": "Informative description of the first subdataset...",
        "license": "https://creativecommons.org/publicdomain/zero/1.0/"
    },
    {
        "@type": "Dataset",
        "name": "Sub dataset 02",
        "description": "Informative description of the second subdataset...",
        "license": "https://creativecommons.org/publicdomain/zero/1.0/"
    }
]
